Count of subsets with Different equal to X
Given an array arr[] of length N and an integer X, the task is to find the number of subsets with Difference equal to X.

Examples:

Input: arr[] = {1, 1, 2, 3}, X = 1
Output: 3
All the possible subsets are 
1. {1, 1, 2}, {3}
2. {1,3}, {2,1}
3. {1,3}, {2,1}


Input: arr[] = {1,1,2,3}, X = 1
Output: 3
